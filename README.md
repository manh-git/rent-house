#  Roommate Finder & Rental Management System

Dự án ứng dụng di động hỗ trợ tìm kiếm phòng trọ, tìm người ở ghép, và kết nối giữa chủ trọ với người đi thuê một cách an toàn, nhanh chóng.

##  Công nghệ sử dụng

###  Frontend (Mobile)
* **Framework:** React Native (TypeScript), Expo Router
* **UI/UX:** TailwindCSS
* **State Management:** Zustand / Redux Toolkit / Context API

### Backend
* **Runtime:** Node.js, Express.js
* **Language:** TypeScript
* **ORM:** Prisma ORM
* **Database:** PostgreSQL

###  Infrastructure & Services
* **Caching/Session:** Redis
* **Real-time:** Socket.io
* **Security:** JWT (JSON Web Tokens), bcrypt
* **Storage:** Cloudinary (Media management)
* **Payment:** VNPay API
* **Containerization:** Docker
